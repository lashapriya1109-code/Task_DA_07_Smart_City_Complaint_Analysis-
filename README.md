# FSI Data Analysis Dashboard
## Problem Statement

The goal of this project is to analyze the Fragile States Index (FSI) dataset and identify global instability patterns, risk levels, and country-wise trends. The project helps understand which countries are most fragile, which are stable, and how global risk has changed over time.

## Dataset Description

Dataset Name: FSI_Data_2006_2023.csv

The dataset contains Fragile States Index scores for multiple countries from 2006 to 2023. It includes various social, economic, and political indicators used to measure the stability of countries.

## Key Findings

* Several countries consistently show high FSI scores and remain highly fragile.
* Stable countries maintain low FSI scores across years.
* Global instability trends vary across different years.
* Some indicators have a strong relationship with overall FSI scores.
